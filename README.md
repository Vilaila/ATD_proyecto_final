# Gasto en educación e inversión en I+D+i y su relación con los premios Nobel: un análisis comparativo por países

## Descripción

Proyecto académico de la asignatura *Adquisición y Transmisión de Datos* cuyo objetivo es adquirir, filtrar y estructurar datos procedentes de fuentes web dinámicas y analizar su relación con el número de premios Nobel por país.

El trabajo se centra principalmente en el uso de **web scraping con Selenium**, **peticiones HTTP** y **procesamiento de texto con expresiones regulares** para la limpieza y normalización de los datos.

---

## Contenido del repositorio

- Notebook de **web scraping** para la obtención de datos de premios Nobel y gasto educativo.
- Notebook de **procesamiento, limpieza y análisis exploratorio** de los datos.
- Archivos CSV generados durante el proceso de adquisición y filtrado.

---

## Tecnologías utilizadas

- Python
- Selenium
- Pandas
- Expresiones regulares (`re`)
- Jupyter Notebook
